## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Cloning project

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/next-tailwind-framermotion.git

  # Enter folder
  $ cd next-tailwind-framermotion

```

## Install and run

```bash

  # Install
  $ yarn

  # Run
  $ yarn dev

```
